# 贡献者行为准则（Contributor Covenant Code of Conduct）

## 我们的承诺

为了营造开放和友善的环境，我们作为贡献者和维护者承诺：让每个人参与本项目和社区的过程，
都能成为不受骚扰的体验——无论年龄、体型、可见或不可见的身心障碍、族裔、性别特征、性别认同与表达、
经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教，或性取向。

我们承诺以有助于建设开放、友善、多元、包容、健康社区的方式行事与互动。

## 我们的准则

有助于营造积极环境的行为包括但不限于：

- 对他人表达同理与善意
- 尊重不同的意见、观点与经验
- 给予并优雅地接受建设性反馈
- 承担责任，向受我们行为影响的人致歉，并从中学习
- 关注的不仅是个人，而是整个社区的最大利益

不可接受的行为包括但不限于：

- 使用带有性意味的言语或图像，以及任何形式的性关注或性挑逗
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可，公布他人的私密信息（如真实姓名、住址或电子邮箱）
- 其他在专业场合中可合理认定为不当的行为

## 执行责任

社区维护者有责任阐明并执行上述可接受行为的准则，并对任何其认为不当、具威胁性、冒犯性或有害的行为，
采取适当且公正的纠正措施。

社区维护者有权移除、编辑或拒绝与本行为准则不符的评论、提交、代码、Wiki 编辑、Issue 及其他贡献，
并在适当时说明理由。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公开场合代表本社区时的情形。代表社区的情形包括但不限于：
使用项目的官方电子邮箱、通过官方社交媒体账号发布，或在线上/线下活动中作为指定代表。

## 执行

如遇辱骂、骚扰或其他不可接受的行为，请通过本仓库的 **GitHub 私密渠道**向负责执行的维护者私下举报：

**`Security` 标签页 → “Report a vulnerability”**，并在标题注明 **`[CoC]`**。这是本项目唯一的私密上报通道，
维护者会在仅双方可见的私密线程中处理；**请勿在公开 Issue / PR 中披露举报内容**。

所有投诉都将得到及时、公正的审查与处理。维护者有义务为举报者保密。

## 执行准则

维护者将依据以下《社区影响准则》，判定其认为违反本行为准则的行为的处理后果：

**1. 纠正（Correction）**
社区影响：使用不当言语或其他在社区中被认定为不专业或不受欢迎的行为。
后果：维护者私下给予书面警告，说明违规性质并解释为何不当；可能要求公开道歉。

**2. 警告（Warning）**
社区影响：单次或一系列行为构成的违规。
后果：给予警告并说明持续行为的后果。在指定时间内，不得与相关人员（含主动联系准则执行者）发生互动；
违反将可能导致临时或永久封禁。

**3. 临时封禁（Temporary Ban）**
社区影响：严重违反社区准则，包括持续的不当行为。
后果：在指定时间内临时禁止与社区进行任何形式的互动或公开交流；违反将可能导致永久封禁。

**4. 永久封禁（Permanent Ban）**
社区影响：表现出违反社区准则的固定模式，包括持续的不当行为、对个人的骚扰，或对特定群体的攻击或贬低。
后果：永久禁止在社区内进行任何形式的公开互动。
